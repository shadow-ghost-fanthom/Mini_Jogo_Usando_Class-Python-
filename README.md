# ⚔️ Mini RPG de Terminal em Python
Jogo de RPG baseado em texto onde o jogador escolhe uma classe e enfrenta monstros aleatórios com desafios de reflexo em tempo real.

🚀 Nota de Aprendizado: Este projeto foi o meu primeiro contato prático com Classes (Programação Orientada a Objetos) no Python! Aprendi a estruturar entidades, passar objetos como argumentos e gerenciar estados de forma organizada.

🕹️ Funcionalidades
🎭 Classes Únicas: Mago, Guerreiro ou Assassino, cada um com HP e ATK próprios.

🎲 Inimigos Aleatórios: Oponentes sorteados a cada partida (O Guardião, Slime Gigante ou Bruxa Sombria).

⚡ Quick Time Event (QTE): Mecânica de desvio onde você precisa digitar a letra exibida na tela antes que o tempo acabe.

🛠️ Como Rodar
O projeto utiliza a biblioteca keyboard para ler os comandos em tempo real.

1. Instale a biblioteca:

```Bash
pip install keyboard
```
2. Execute o jogo:
```Bash
   python nome_do_seu_arquivo.py
```
